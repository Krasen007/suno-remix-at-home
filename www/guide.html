🚀 Real-World Usage Guide

Everything you need to start using Suno Remix at Home effectively in production or daily use.

� 5-Minute Quick Start

1️⃣ Start the Server

python server.py

Open your browser and visit http://localhost:5000

2️⃣ Add Your API Key

The first time you visit, you'll see a welcome modal. Just paste your Suno API key and click Save.

💡 Pro Tip: Your key is stored locally and never sent to any server.

3️⃣ Add Your Audio

Paste any public audio URL (GitHub, Dropbox, etc.) or use the upload button for GitHub integration.

4️⃣ Start Remixing

Click "🎵 Start Remix" and watch the real-time progress in your browser.

🔑 Understanding API Keys

✅ What You Need

  • Just a Suno API key from sunoapi.org
  • No GitHub token required for basic usage
  • Browser storage keeps your key private and secure

❌ What You Don't Need

  • Environment variables (unless you're an advanced user)
  • Server configuration files
  • GitHub token for basic remixing
  • Complex setup processes

🎵 Audio Hosting Options

🌐 Option 1: Any Public URL

Upload to Dropbox, Google Drive, personal website, or any public hosting service.

1. Upload file anywhere
2. Copy public URL
3. Paste in "Audio URL" field

🐙 Option 2: GitHub Integration

Automatic hosting with GitHub integration (optional but convenient).

1. Create GitHub token
2. Add to .env file
3. Use upload button in interface

💡 Note: This is optional - you can use any hosting service!

💾 Option 3: Local Files Only

Process files locally without any hosting setup.

1. Leave "Audio URL" field empty
2. Files save to remixes/ folder
3. Manual download when complete

Process multiple tracks in parallel for faster workflows.

Your browser localStorage keeps history forever - no server setup needed.
